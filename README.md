# Painting Fever

A music-based scroller platformer where you paint the walls as you navigate through rhythmic levels.

## Version
**Alpha 0.1**

## About
Painting Fever is a 2D platformer that combines music-driven gameplay with creative wall painting mechanics. Navigate through dynamic levels while painting your way to victory.

## Current Status
This game is in early development. Currently implemented features include basic platforming and movement mechanics.

**Features in development:**
- Pause menu
- Game end screen
- Wall painting mechanics
- Level progress system
- Music BPM action-sync

## Built With
- Godot 4.5.1 (2D)
